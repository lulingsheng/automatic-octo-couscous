# 🚀 快速开始指南 - 豆包 AI 版本

## ⚡ 5 分钟快速启动

### 步骤 1: 安装依赖（1 分钟）

```bash
pip install -r requirements.txt
```

### 步骤 2: 测试集成（1 分钟）

```bash
python test_doubao_integration.py
```

看到 `✅ 所有测试完成!` 表示集成成功！

### 步骤 3: 启动应用（1 分钟）

```bash
streamlit run app.py
```

### 步骤 4: 登录系统（30 秒）

- 用户名: `admin`
- 密码: `123456`

### 步骤 5: 体验 AI 功能（1.5 分钟）

1. 点击侧边栏 **"AI智能问答"**
2. 在聊天框输入: **"画出企业ROA的柱状图"**
3. 观看 AI 生成图表！

## 🐛 常见问题

### Q: PySpark 初始化失败？
**A**: 正常现象，系统会自动降级到 Pandas，不影响使用。

### Q: 图表不显示？
**A**: 检查数据列名是否正确，确认数据集已加载。

### Q: API 调用失败？
**A**: 系统会自动降级到 Mock 模式，不影响使用。
